JQuery Selectors

JQuery allows easy selection of html elements

  • JQuery is a Javascript library
  • In JQuery, first we select something, then apply an action to it
  • important: Do not forget to reference the JQuery library
  • Majority of the time, a JQuery selector returns a JQuery object
  • JQuery actions can be chained

JQuery's simplicity in selecting many items and then be able to apply actions is making it one of the most popular Javascript libraries.

JQuery has excellent support for AJAX operations

JQuery selectors can be divided into 5 main categories
